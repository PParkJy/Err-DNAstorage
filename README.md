# Error-DNAstorage
This repository is for the datasets used to investigate the error statistics of DNA storage.    

We provide of cif files and FASTQ files of tile 1101.    
The cif files are stored in folders of each cycle.    
The FASTQ file of R1 and R2 reads is provided, and the files was generated by basecaller [AYB](https://www.ebi.ac.uk/research/goldman/software/AYB).
